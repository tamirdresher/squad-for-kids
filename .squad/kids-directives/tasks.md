## How to Add a Task

Add a row to the **Active Tasks** table above. Format:
- **ID:** PT-{sequential number}
- **Status options:** `⏳ pending`, `🔄 in progress`, `✅ done`, `⏸️ paused`

Kids' Ralph will create a GitHub issue for each `⏳ pending` task it hasn't seen before.
When the issue is closed, Ralph updates the status to `✅ done` here.
