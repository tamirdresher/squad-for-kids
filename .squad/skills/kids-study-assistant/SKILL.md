# Kids Study Assistant — Squad Skill

> מעקב אחרי מבחנים ויצירת תוכניות לימוד יומיות

## סקירה

ה-Kids Study Assistant הוא כלי שעוזר לנהל לוחות מבחנים ויוצר תוכניות לימוד מובנות. הוא מנתח נתוני מבחנים מ-YAML, מייצר לוחות לימוד יומיים עם חזרה מרווחת, ומייצא תוכניות ל-Markdown.

## יכולות

- **ניתוח לוח מבחנים** מקבצי YAML
- **חישוב ימים עד מבחנים** עם אינדיקטורים (דחוף, קרוב, עתידי)
- **יצירת תוכניות לימוד יומיות** עם הקצאת נושאים וחזרה מרווחת
- **אי-הכללת שבת** (שישי ערב עד מוצאי שבת)
- **ייצוא ל-Markdown** לשיתוף קל
- **תמיכה במספר תלמידים** — מעקב אחרי כמה ילדים
- **תמיכה בעברית** (RTL)

## שימוש

```powershell
cd kids-study

# הצגת לוח מבחנים
./Get-ExamSchedule.ps1

# יצירת תוכנית לימוד
./New-StudyPlan.ps1 -DaysAhead 14 -ExcludeShabbat

# ייצוא התוכנית להיום
./Export-StudyPlan.ps1

# הפעלת שגרה יומית מלאה
./Start-DailyStudyRoutine.ps1
```

## הגדרות

ערכו את `schedule.yaml`:

```yaml
students:
  - name: "שם הילד"
    grade: "כיתה ז'"
    exams:
      - subject: "מתמטיקה"
        date: "2026-03-25"
        topics:
          - "משוואות"
          - "שברים"
```

## דוגמת פלט

```markdown
# 📚 תוכנית לימוד יומית
**תאריך:** יום ראשון, 22 מרץ 2026

## ⚠️ מבחנים קרובים
- 🎯 מתמטיקה בעוד **3 ימים** (25.3.2026)

## 📖 שיעורי היום
| שעה | מקצוע | נושא | משך | סוג |
|-----|--------|------|-----|-----|
| 16:00 | מתמטיקה | משוואות | 20 דק' | חזרה |
| 16:30 | מתמטיקה | שברים | 20 דק' | תרגול |
```
