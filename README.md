## 🇮🇱 עברית

### מה זה Kids Squad?

Kids Squad זה **צוות AI שעובד בשבילך**. במקום עוזר אחד, יש לך צוות שלם:
- 🎓 **מורה** שמסביר מושגים חדשים
- 💻 **מתכנת** שכותב קוד איתך
- 🎨 **מעצב** שעוזר עם עיצוב
- 🧪 **בודק** שבודק שהכל עובד
- 📚 **חוקר** שעוזר בהכנה למבחנים

### 🚀 איך מתחילים? (30 שניות!)

#### שלב 1: Fork
לחץ על כפתור **Fork** למעלה ↗️

#### שלב 2: פתח סביבת פיתוח
בחר אחת מהאפשרויות:

**אפשרות א' — Codespace (מומלץ! בלי להתקין כלום):**
1. לחץ על **Code** → **Codespaces** → **Create codespace on main**
2. חכה דקה שהסביבה תיטען ⏳

**אפשרות ב' — VS Code מקומי:**
1. Clone את ה-repo למחשב שלך
2. פתח ב-VS Code
3. התקן את התוסף GitHub Copilot

#### שלב 3: שלום!
1. פתח את **Copilot Chat** (הצ'אט למעלה מימין, או `Ctrl+Shift+I`)
2. כתוב: **שלום**
3. ה-AI ינחה אותך צעד אחר צעד! 🎉

#### שלב 4: השתמש בצוות!
```
💬 "צוות: תעזרו לי עם שיעורי בית במתמטיקה"
💬 "מורה: תסביר לי מה זה משתנים"
💬 "מתכנת: תעזור לי לבנות משחק"
💬 "חוקר: יש לי מבחן בהיסטוריה מחר"
```

## 🇺🇸 English

### What is Kids Squad?

Kids Squad is an **AI team that works for you**. Instead of one assistant, you get a whole team:
- 🎓 **Teacher (מורה)** who explains new concepts
- 💻 **Programmer (מתכנת)** who codes with you
- 🎨 **Designer (מעצב)** who helps with design
- 🧪 **Tester (בודק)** who checks everything works
- 📚 **Researcher (חוקר)** who helps prep for exams

### 🚀 Getting Started (30 seconds!)

1. **Fork** this repo (button above ↗️)
2. **Open** in GitHub Codespace or VS Code with Copilot
3. **Type** `שלום` (or `hello`) in Copilot Chat
4. **Follow** the interactive setup guide

### How It Works

The repo comes pre-configured with:
- `.github/copilot-instructions.md` — Makes Copilot respond in Hebrew and guide setup
- `.github/agents/squad.agent.md` — Kid-friendly team coordinator
- `.devcontainer/` — Zero-install development environment
- `starter-projects/` — Age-appropriate project templates
- `ralph-kids.ps1` — Project monitoring companion (Hebrew!)
- `setup-notifications.ps1` — WhatsApp/Discord notification setup

### Age-Adaptive

| Age | Team Size | Focus | Language |
|-----|-----------|-------|----------|
| 8-10 | 3 agents | Fun projects, learning | Simple Hebrew + emojis |
| 11-13 | 4 agents | Games, websites, bots | Clear Hebrew + tech terms |
| 14+ | 5+ agents | Full-stack, exam prep, data | Professional Hebrew |
