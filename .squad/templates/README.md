## Template Format

Each template file defines:
- **Squad name and description**
- **Target age range**
- **Agent roster** with personalities and specialties
- **System prompts** for each agent
- **Interaction guidelines** and safety rules
- **Gamification settings**

## Creating Your Own Template

1. Copy an existing template as a starting point
2. Define your agents (3-6 recommended)
3. Write system prompts that establish personality and boundaries
4. Set age-appropriate content guidelines
5. Submit for safety review before publishing
