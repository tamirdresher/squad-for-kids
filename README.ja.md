## 問題

どの教室にも25〜35人の生徒、1人の先生、1つのペースがあります。進みが速い子は退屈してしまい、遅れた子は自信を失います。保護者は助けたいと思っていますが、すべての教科を教える時間も専門知識も忍耐もありません。家庭教師は1時間5,000〜10,000円—ほとんどの家庭には手が届きません。

**結果は？** 何百万人もの子どもたちが「算数が苦手」「読書が嫌い」と思いながら育ちます—本当は、*自分に合った方法*で説明してくれる人が必要なだけなのに。

## 解決策

**キッズ・スクワッド**は、すべての子どもに独自のAI学習スペシャリストチームを提供します。それぞれがユニークな性格、教え方、専門分野を持っています。お子さまの年齢、興味、ペースに合わせて適応します。成功を一緒に喜び、困難をサポートし、学びを遊びのように楽しくします。

まるで個人専用の学校—天使のような忍耐力、YouTuberのような創造性、そして素晴らしい先生のような心を持っています。

### 仕組み

1. 📝 **名前を教えてね** — チームがお子さまを個人的にお出迎え
2. 🎂 **学年を入力** — 正しいカリキュラムに自動マッチング
3. 🏙️ **住んでいる街を共有** — 国、教育システム、言語を自動検出
4. 📚 **カリキュラムがロード** — 米国Common Core、英国ナショナル・カリキュラム、イスラエルBagrutなど
5. 🎬 **好きな世界を選ぼう** — エージェントがお気に入りのアニメやゲームのキャラクターに変身！

## ✨ 主な機能

### 🎯 年齢適応型指導
難易度が自動的に調整されます。6歳の子には簡単な言葉と絵。12歳の子にはより深い説明とチャレンジ。チームはお子さまと一緒に成長します。

### 💡 興味に基づく学習
お子さまが興味を選びます—恐竜、宇宙、マインクラフト、料理、サッカー—チームがそれをすべてのレッスンに織り込みます。

### 📊 進捗追跡と保護者レポート
保護者は週次レポートを受け取ります：
- カバーしたトピックと学習時間
- 得意分野と改善が必要な分野
- 週末のおすすめ活動
- 心の健康指標

### 🛡️ 安全とプライバシー
- **個人データの保存なし** — 会話はお使いのデバイスに留まります
- **年齢に適したコンテンツフィルタリング** — 各年齢層に合わせた保護
- **会話の安全性** — エージェントが不適切な話題をやさしくリダイレクト
- **オープンソース** — すべてのコードを自分で確認できます

### 🎮 ゲーミフィケーション
- **XPポイント** — レッスン完了で経験値を獲得
- **レベル** — 初級、中級、上級へと進級
- **バッジ** — 実績を集めよう（「数学の魔法使い」「本の虫」「科学探検家」）
- **連続記録** — 毎日の学習ストリークでボーナス報酬

### 🤗 感情的サポート
学習バディがお子さまのフラストレーション、退屈、不安を察知し、共感を持って対応します。

### 🔊 読み上げ機能
小さなお子さまや聴覚型学習者向けに、テキスト読み上げ技術でレッスンを音声に変換できます。

### 📅 学習スケジューラー
自動学習リマインダー、宿題追跡、間隔反復スケジューリング。

### 🌍 対応カリキュラム

| 国 | カリキュラム | 学年 |
|----|-----------|------|
| 🇯🇵 日本 | 学習指導要領 | 小1〜高3 |
| 🇺🇸 アメリカ | Common Core State Standards | K–12 |
| 🇬🇧 イギリス | National Curriculum | Year 1–13 |
| 🇮🇱 イスラエル | 教育省カリキュラム | 1〜12年生 |
| 🇦🇺 オーストラリア | Australian Curriculum (ACARA) | Foundation–Year 12 |
| 🇨🇦 カナダ | Provincial Curricula | Grade 1–12 |
| 🌍 どの国でも | ウェブ検索で自動検出 | 様々 |

### 👪 保護者コントロール
- 1日/1週間の時間制限を設定
- 会話履歴を確認
- 特定のトピックを承認またはブロック
- 難易度を調整

## 🚀 保護者向けクイックスタート

### ステップ1：チームを入手

```bash
git clone https://github.com/tamirdresher/squad-for-kids.git
cd squad-for-kids
```

### ステップ2：初回セットアップ（お子さまがやります！）

`.github/agents/squad.agent.md` のエージェントでCopilotセッションを開始。システムは：

1. 🎉 **お出迎え** — 温かくて楽しい挨拶
2. 📝 **名前を聞く** — パーソナライズに使用
3. 🎂 **誕生日を聞く** — 年齢計算のため
4. 🏫 **学年を聞く** — 正しいカリキュラムにマッチング
5. 🏙️ **街を聞く** — 国とカリキュラムを検出
6. 🗣️ **言語を設定** — デフォルトは国の言語
7. 🎬 **好きな映画/アニメ/ゲームを聞く** — エージェントがお気に入りのキャラクターに変身！

### ステップ3：学習スタート！

お子さまはパーソナライズされたAIチューターチームを受け取ります：
- カリキュラムのどの教科でも学習
- 宿題のヘルプ（ガイド付き、答えだけではなく）
- 教育ゲームでXPを稼ぐ
- セッション間に休憩を取る
