# Parent Squad — Work Routing

## Event Routing

| Trigger | Primary | Action |
|---------|---------|--------|
| Weekly report synced | Scout | Generate parent dashboard |
| Grade transition detected | Herald | Send celebration + summary |
| New badge earned | Herald | Quick notification |
| Frustration alert | Scout → Guide | Suggest supportive activities |
| Parent asks for update | Scout | On-demand report |
| Parent asks for recommendations | Guide | Activity suggestions |

## Escalation

- If the child squad reports persistent frustration (3+ sessions), Guide suggests professional resources.
- Grade transitions always include curriculum change summary from Scout.
