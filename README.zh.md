## 问题

每个教室有 25-35 名学生、一位老师和统一的进度。有些孩子进度太快会感到无聊，有些落后后会失去信心。家长想要帮忙，但缺乏时间、专业知识或耐心来辅导每一门课程。私人辅导费用高达每小时 200-500 元——大多数家庭难以承受。

**结果呢？** 数百万孩子在成长过程中认为自己"数学不好"或"不爱阅读"——其实他们只是需要有人用*他们的方式*来讲解。

## 解决方案

**儿童学习小队** 为每个孩子提供一个完整的 AI 学习专家团队——每位专家都有独特的性格、教学风格和专业领域。他们会根据孩子的年龄、兴趣和进度进行调整。他们庆祝成功、帮助克服困难，让学习充满乐趣。

这就像拥有一所私人学校——拥有天使般的耐心、YouTuber 般的创意和优秀教师般的爱心。

### 工作原理

1. 📝 **告诉我们你的名字** — 团队会亲切地问候你的孩子
2. 🎂 **输入年级** — 自动匹配正确的课程体系
3. 🏙️ **分享你的城市** — 自动检测国家、课程体系和语言
4. 📚 **加载课程** — 支持美国 Common Core、英国国家课程、以色列 Bagrut 等
5. 🎬 **选择你的世界** — AI 助手会变成孩子最喜欢的动画或游戏中的角色！

## ✨ 核心特性

### 🎯 年龄自适应教学
难度级别自动调整。6 岁的孩子会看到简单的文字和图片。12 岁的孩子会得到更深入的解释和挑战。团队会与您的孩子一起成长。

### 💡 兴趣驱动学习
您的孩子选择自己的兴趣——恐龙、太空、我的世界、烹饪、足球——团队会将这些融入每一节课。

### 📊 进度追踪与家长报告
家长每周收到报告，包括：
- 已覆盖的主题和花费的时间
- 优势领域和需要改进的地方
- 周末活动建议
- 情绪健康指标

### 🛡️ 安全与隐私
- **不存储个人数据** — 对话保留在您的设备上
- **适龄内容过滤** — 为每个年龄段提供保护
- **对话安全** — 助手会温和地引导不当话题
- **开源** — 您可以自己检查每一行代码
- **最少数据收集** — 仅需名字、出生日期和城市

### 🎮 游戏化
- **经验值** — 完成课程获得经验
- **等级** — 从初级到中级到高级
- **徽章** — 收集成就（"数学魔法师"、"书虫"、"科学探索者"）
- **连续学习** — 保持每日学习连续获得额外奖励

### 🤗 情感支持
学习伙伴能察觉孩子何时感到沮丧、无聊或焦虑，并以同理心回应。

### 🔊 朗读功能
对于年幼的孩子或听觉型学习者，课程可以通过文字转语音技术转换为音频。

### 📅 学习计划器
自动学习提醒、作业追踪和间隔复习安排。

### 🌍 支持的课程体系

| 国家 | 课程体系 | 年级 |
|------|---------|------|
| 🇺🇸 美国 | Common Core State Standards | K–12 |
| 🇬🇧 英国 | National Curriculum | Year 1–13 |
| 🇮🇱 以色列 | 教育部课程 | 1–12 年级 |
| 🇨🇳 中国 | 义务教育课程标准 | 1–12 年级 |
| 🇦🇺 澳大利亚 | Australian Curriculum (ACARA) | Foundation–Year 12 |
| 🇨🇦 加拿大 | Provincial Curricula | Grade 1–12 |
| 🇮🇳 印度 | CBSE / ICSE / State Boards | Class 1–12 |
| 🌍 任何国家 | 通过网络搜索自动检测 | 各异 |

### 👪 家长控制
- 设置每日/每周时间限制
- 查看对话历史
- 批准或屏蔽特定主题
- 调整难度级别

## 🚀 家长快速入门

### 第 1 步：获取团队

```bash
git clone https://github.com/tdsquadAI/squad-for-kids.git
cd squad-for-kids
```

### 第 2 步：首次设置（孩子来完成！）

使用 `.github/agents/squad.agent.md` 中的代理启动 Copilot 会话。系统将：

1. 🎉 **欢迎他们** — 温暖有趣的问候
2. 📝 **询问名字** — 用于个性化
3. 🎂 **询问生日** — 计算年龄
4. 🏫 **询问年级** — 匹配正确课程
5. 🏙️ **询问城市** — 检测国家和课程
6. 🗣️ **设置语言** — 默认为该国语言
7. 🎬 **询问最喜欢的电影/动画/游戏** — 助手变成他们最爱的角色！

### 第 3 步：开始学习！

您的孩子将获得个性化的 AI 导师团队。他们可以：
- 学习课程中的任何科目
- 获得作业帮助（引导式，绝不只是答案）
- 玩教育游戏并赚取经验值
- 在学习之间休息
