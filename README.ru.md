## Проблема

В каждом классе 25-35 учеников, один учитель и единый темп. Одни дети опережают программу и скучают. Другие отстают и теряют уверенность. Родители хотят помочь, но им не хватает времени, знаний или терпения, чтобы объяснять каждый предмет. Репетиторы стоят 2000-5000₽/час — большинству семей это не по карману.

**Результат?** Миллионы детей вырастают, думая, что они «плохи в математике» или «не любят читать» — а им просто нужен был кто-то, кто объяснит *их способом*.

## Решение

**Сквад для Детей** даёт каждому ребёнку собственную команду ИИ-специалистов по обучению — каждый с уникальной личностью, стилем преподавания и специализацией. Они адаптируются к возрасту, интересам и темпу вашего ребёнка. Празднуют победы, помогают справляться с трудностями и делают обучение похожим на игру.

Это как персональная школа — с терпением ангела, креативностью блогера и сердцем великого учителя.

### Как это работает

1. 📝 **Назови своё имя** — команда приветствует ребёнка лично
2. 🎂 **Укажи класс** — автоматически подбирается нужная учебная программа
3. 🏙️ **Назови свой город** — автоматически определяет страну, учебную систему и язык
4. 📚 **Загружается программа** — Common Core (США), Национальная программа (Великобритания), Багрут (Израиль) и другие
5. 🎬 **Выбери свою вселенную** — агенты превращаются в персонажей из любимого мультфильма или игры!

## ✨ Ключевые возможности

### 🎯 Адаптивное обучение по возрасту
Уровень сложности настраивается автоматически. 6-летний получает простые слова и картинки. 12-летний — более глубокие объяснения и задания. Команда растёт вместе с вашим ребёнком.

### 💡 Обучение через интересы
Ребёнок выбирает свои интересы — динозавры, космос, Minecraft, кулинария, футбол — и команда вплетает их в каждый урок.

### 📊 Отслеживание прогресса и отчёты для родителей
Родители получают еженедельный отчёт:
- Пройденные темы и затраченное время
- Сильные стороны и области для улучшения
- Предложенные занятия на выходные
- Показатели эмоционального состояния

### 🛡️ Безопасность и конфиденциальность
- **Никаких личных данных** — разговоры остаются на вашем устройстве
- **Фильтрация контента по возрасту** — защита для каждой возрастной группы
- **Безопасность общения** — агенты мягко перенаправляют неподходящие темы
- **Открытый исходный код** — проверьте каждую строку кода сами

### 🎮 Геймификация
- **Очки XP** — зарабатывай опыт за выполнение уроков
- **Уровни** — продвигайся от новичка через средний к продвинутому
- **Значки** — собирай достижения («Математический маг», «Книжный червь», «Научный исследователь»)
- **Серии** — поддерживай ежедневные серии обучения для бонусов

### 🤗 Эмоциональная поддержка
Учебный товарищ определяет, когда ребёнок расстроен, скучает или тревожится, и реагирует с сочувствием.

### 🔊 Чтение вслух
Для маленьких детей или аудиальных учеников уроки можно преобразовать в аудио с помощью технологии озвучивания текста.

### 📅 Планировщик занятий
Автоматические напоминания о занятиях, отслеживание домашних заданий и планирование интервального повторения.

- Установка дневных/недельных лимитов времени

## 🚀 Быстрый старт для родителей

### Шаг 1: Получить команду

```bash
git clone https://github.com/tdsquadAI/squad-for-kids.git
cd squad-for-kids
```

### Шаг 2: Первоначальная настройка (Ваш ребёнок делает это!)

Запустите сессию Copilot с агентом в `.github/agents/squad.agent.md`. Система:

1. 🎉 **Поприветствует** — тёплое, весёлое приветствие
2. 📝 **Спросит имя** — для персонализации
3. 🎂 **Спросит дату рождения** — для расчёта возраста
4. 🏫 **Спросит класс** — подберёт правильную программу
5. 🏙️ **Спросит город** — определит страну и программу
6. 🗣️ **Установит язык** — по умолчанию язык страны
7. 🎬 **Спросит любимый фильм/мультик/игру** — агенты станут персонажами из любимой вселенной!

### Шаг 3: Обучение начинается!

Ваш ребёнок получает персонализированную команду ИИ-репетиторов:
- Изучение любого предмета из программы
- Помощь с домашними заданиями (руководство, а не готовые ответы)
- Образовательные игры с начислением XP
- Перерывы между занятиями
