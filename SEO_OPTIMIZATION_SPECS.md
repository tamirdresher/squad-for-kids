## Performance Optimizations (For Core Web Vitals)

1. **Lazy load images** - Use `loading="lazy"` attribute
2. **Minimize CSS/JS** - Remove unused stylesheets
3. **Use WebP images** - Modern image format for blog post images
4. **Compress images** - Target <200KB per image
5. **Enable gzip compression** - Server-side (GitHub Pages)
6. **Reduce render-blocking resources** - Defer non-critical CSS/JS
