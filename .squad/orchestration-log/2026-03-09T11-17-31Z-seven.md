### 3. Separate Multimodal and Podcaster Agents

**Rationale:** Completely different tool chains, skills, routing rules
- **Multimodal (Geordi):** Mermaid-CLI, D2, image generation, Playwright
- **Podcaster (NEW agent):** Azure AI Speech, TTS, audio file handling

**Combining them:** Creates bloated, unfocused agent

### 4. Code-Based Diagrams Over AI Image Generation

**Advantages of Mermaid:**
- Version-controllable, diffable, GitHub-native
- Any LLM can generate code (no special model needed)
- Renders reliably in CI/CD (SVG/PNG via mermaid-cli)

**Disadvantages of AI images:**
- Non-deterministic (can't incrementally edit in code)
- Requires API calls + cost

**Decision:** Use Mermaid as primary diagram tool. Reserve image generation for creative/illustrative tasks only.

### 5. Model Leveraging

- ✅ No new models needed for primary use case (Mermaid code)
- ⚠️ True image generation requires GPT-4o API calls (cost consideration)
- ⚠️ Gemini Flash image gen would require catalog addition
