אפשרות א׳: Codespace — הדרך הקלה ביותר

אין צורך להתקין כלום. הכול רץ בענן, ישירות בדפדפן.

צרו חשבון GitHub

אם עדיין אין לכם חשבון, היכנסו ל-github.com/signup וצרו חשבון חינמי.

פתחו מחשב נייד או נייח

⚠️ חשוב: יש לבצע את כל השלבים ממחשב נייד או נייח (לא מטלפון!). הטלפון לא תומך ב-Codespaces באופן מלא.

היכנסו לריפוזיטורי וצרו Codespace

עברו לכתובת:

→ github.com/tamirdresher/squad-for-kids
  • לחצו על הכפתור הירוק Code
  • עברו ללשונית Codespaces
  • לחצו Create codespace on main

המתינו 1-2 דקות

סביבת עבודה מלאה (VS Code בדפדפן) תיטען אוטומטית. תראו עורך קוד עם חלון טרמינל בתחתית.

פתחו את Copilot Chat

לחצו על אייקון הצ׳אט 💬 בסרגל הצד, או השתמשו בקיצור:

Ctrl + Alt + I

בחרו את הסוכן "squad"

בתחתית חלון הצ׳אט, לחצו על התפריט ובחרו את הסוכן squad מהרשימה.

הפעילו Autopilot

לחצו על הכפתור Autopilot (Preview) בחלון הצ׳אט. זה מאפשר לסוכן לפעול באופן עצמאי.

תנו לילד/ה להתחיל! 🎉

כתבו בעברית! אפשר להתחיל עם:

💬 היי! אני רוצה לבנות משחק!

הסוכן ידריך את הילד/ה צעד אחר צעד.

💡 חזרה לעבודה קיימת: נכנסתם בעבר ורוצים להמשיך? פשוט גשו ל-github.com/codespaces ותמצאו את סביבת העבודה הקיימת שלכם.
📖 מילון מונחים — תרגום כפתורים מאנגלית

ממשק GitHub הוא באנגלית. הנה תרגום של הכפתורים והמונחים שתפגשו:

מה רואים (אנגלית) מה זה אומר (עברית)

Codespaces לשונית סביבת העבודה בענן

🍴 אפשרות ב׳: Fork — העותק שלכם

שליטה מלאה: כל הקוד, ההיסטוריה והדוחות — ברשות שלכם.

מה זה Fork? Fork הוא עותק מלא של הפרויקט שנמצא בחשבון GitHub שלכם. כל מה שהילד/ה יוצר/ת — קוד, פרויקטים, דוחות התקדמות — נשמר אצלכם. אתם הבעלים.

צרו Fork

גשו ל-github.com/tamirdresher/squad-for-kids ולחצו על כפתור Fork בפינה הימנית העליונה.

ודאו שאתם ב-Fork שלכם

אחרי ה-Fork, בדקו שהכתובת בדפדפן מראה:

✅ github.com/YOUR-USERNAME/squad-for-kids

ולא:

// ❌ לא זה!
github.com/tamirdresher/squad-for-kids

צרו Codespace מה-Fork שלכם

כמו באפשרות א׳ — לחצו Code → Codespaces → Create codespace on main, אבל מתוך ה-Fork שלכם.

בחרו squad + Autopilot

פתחו Copilot Chat, בחרו את הסוכן squad והפעילו Autopilot.

הילד/ה מתחיל/ה לכתוב! 🚀

בדיוק כמו באפשרות א׳ — רק שעכשיו הכול שמור בחשבון שלכם.

👀 מה ההורים יכולים לראות?
📋

פרופיל הלומד

קובץ student-profile.json מכיל את ההעדפות, הרמה, והנושאים שהילד/ה למד/ה.

📊

דוחות התקדמות

תיקיית .squad/reports/ מכילה דוחות שבועיים על הפעילות והלמידה.

📝

תוכנית לימודים

קובץ .squad/teaching-plan.md מתאר את התוכנית המותאמת אישית.

📜

היסטוריית שינויים

כל commit ב-Git שומר מה השתנה, מתי, ואיך — תיעוד מלא של הדרך.

🔄 עדכון ה-Fork (סנכרון עם המקור)

מדי פעם, הפרויקט המקורי מתעדכן עם שיפורים חדשים. כדי לקבל אותם:

  1. גשו ל-Fork שלכם ב-GitHub
  2. אם יש הודעה This branch is X commits behind — לחצו Sync fork
  3. לחצו Update branch
💡 טיפ: העדכון לא ימחק את העבודה של הילד/ה. הוא רק מוסיף את השיפורים החדשים מהפרויקט המקורי.

💻 אפשרות ג׳: התקנה על מחשב הילד (Windows)

עבודה מקומית עם GitHub Desktop — פשוט וידידותי.

הורידו GitHub Desktop

גשו ל-desktop.github.com והורידו את האפליקציה.

התחברו לחשבון GitHub

פתחו את GitHub Desktop ובצעו Sign In עם חשבון GitHub שלכם.

שכפלו (Clone) את הפרויקט

בחרו Clone a repository וחפשו:

tamirdresher/squad-for-kids

בחרו תיקיית יעד במחשב ולחצו Clone.

הורידו Node.js

גשו ל-nodejs.org והורידו את הגרסה ה-LTS (המומלצת).

הורידו GitHub Copilot CLI

עקבו אחר ההוראות ב-docs.github.com/en/copilot להתקנת Copilot CLI.

פתחו טרמינל בתיקיית הפרויקט

לחצו קליק ימני על תיקיית הפרויקט ובחרו "Open in Terminal" (או "פתח בטרמינל").

הפעילו את Copilot

בטרמינל, כתבו:

$ copilot

ולחצו Enter.

בחרו את הסוכן "Squad"

מתוך הרשימה שתופיע, בחרו Squad.

תנו לילד/ה את המקלדת! ⌨️

הילד/ה יכול/ה לכתוב בעברית ולהתחיל לבנות.

🔨 למתקדמים

אפשרות ד׳: התקנה מתקדמת (VS Code)

התקנה מלאה עם VS Code — שליטה מקסימלית.

🪟 Windows

התקינו VS Code

הורידו מ-code.visualstudio.com והתקינו.

התקינו GitHub Copilot Extension

ב-VS Code, לחצו על אייקון ה-Extensions (או Ctrl+Shift+X), חפשו GitHub Copilot ולחצו Install.

התקינו Git

הורידו מ-git-scm.com והתקינו עם ההגדרות ברירת המחדל.

שכפלו את הפרויקט

פתחו טרמינל ב-VS Code (Ctrl+`) והריצו:

$ git clone https://github.com/tamirdresher/squad-for-kids.git
$ cd squad-for-kids

פתחו את התיקייה ב-VS Code

File → Open Folder → בחרו את תיקיית squad-for-kids.

פתחו Copilot Chat ובחרו squad

לחצו Ctrl+Alt+I, בחרו squad מהתפריט, הפעילו Autopilot — והתחילו!

🍎 macOS

התקינו VS Code

הורידו מ-code.visualstudio.com. גררו לתיקיית Applications.

התקינו GitHub Copilot Extension

ב-VS Code, לחצו ⌘+Shift+X, חפשו GitHub Copilot ולחצו Install.

התקינו Git

פתחו Terminal והריצו:

$ xcode-select --install

זה יתקין את Git (ו-Xcode Command Line Tools).

שכפלו את הפרויקט

בטרמינל של VS Code (⌘+`):

$ git clone https://github.com/tamirdresher/squad-for-kids.git
$ cd squad-for-kids

פתחו את התיקייה ב-VS Code

File → Open Folder → בחרו את תיקיית squad-for-kids.

פתחו Copilot Chat ובחרו squad

לחצו ⌘+Alt+I, בחרו squad, הפעילו Autopilot — ויאללה!

👨‍👩‍👧 ניהול וצפייה בהתקדמות

איך לעקוב אחרי מה שהילד/ה עושה — בלי להפריע.

📊 דוח הורים

ה-Squad מייצר אוטומטית דוחות התקדמות. כדי לראות אותם:

  1. גשו ל-Fork שלכם ב-GitHub
  2. נווטו לתיקייה .squad/reports/
  3. כל דוח מכיל: נושאים שנלמדו, פרויקטים שנבנו, רמת הקושי, והמלצות
✨ טיפ: הדוחות נכתבים בעברית וכתובים בשפה שההורים יבינו — לא צריך ידע טכני כדי לקרוא אותם.
📌 לוח פרויקטים (GitHub Project Board)

ניתן להגדיר לוח פרויקט (Project Board) ב-GitHub כדי לעקוב אחרי משימות:

  • To Do — משימות שמחכות
  • In Progress — מה שהילד/ה עובד/ת עליו עכשיו
  • Done — מה שהושלם

זה מאפשר לילד/ה ללמוד ניהול פרויקטים — ולהורים לראות סטטוס בזמן אמת.
