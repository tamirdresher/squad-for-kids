Getting Started in 3 Minutes

🌟 Recommended

The Easiest Way

  1. Have a GitHub account? Great! If not — create a free account (takes 1 minute, just like signing up for any website)
  2. 🖥️ Important: This step must be done from a laptop or desktop — not a phone or tablet.
    Go to github.com/tamirdresher/squad-for-kids
    Click the green "Code" button → "Codespaces" tab → "Create codespace on main"
    ⏱️ Wait 1–2 minutes for the VS Code environment to fully load.
  3. Wait until the VS Code interface is fully loaded (you'll see a code editor with files on the side)
  4. Open Copilot Chat — click the chat icon 💬 on the side, or press Ctrl+Alt+I
  5. ⚡ Important! At the bottom of the chat window, select the "squad" agent from the dropdown menu
    🔍 Look at the bottom of the chat panel for a dropdown → select squad
  6. Click "Autopilot (Preview)" at the bottom of the chat window to enable autonomous mode — this lets the Squad work on its own!
  7. Start chatting! 🎮 Try: Hey! I want to build a game!
🔄 Coming back later? No need to create a new one! Go to github.com/codespaces and you'll find your existing workspace. Just click on it.

💻 Install on Your Child's Computer (Windows)

Prefer everything local on your child's machine? Here's how:

  1. Download and install GitHub Desktop — click "Download for Windows"
  2. Sign in with your GitHub account
  3. Click "Clone a Repository" → type: tamirdresher/squad-for-kids → Clone
  4. Download and install Node.js (click the big green button)
  5. Download and install GitHub Copilot CLI
  6. Open the folder → right-click → "Open in Terminal"
  7. Type: copilot and press Enter
  8. ⚡ Select the "Squad" agent from the Agent menu at the top
  9. Hand the keyboard to your child! 🎮

🪟 Windows

  1. Download and install Visual Studio Code (free software by Microsoft)
  2. Open VS Code, click the Extensions icon on the left sidebar, search for "GitHub Copilot" — install it
  3. Download Git and install (click Next through the wizard)
  4. Open Terminal inside VS Code (menu: Terminal → New Terminal)
  5. Type: git clone https://github.com/tamirdresher/squad-for-kids.git
  6. Open the created folder: File → Open Folder → squad-for-kids
  7. In the Terminal type: copilot and press Enter
  8. ⚡ Select the "Squad" agent from the menu — that's what activates the learning team

🍎 Mac

  1. Download and install Visual Studio Code
  2. Install the "GitHub Copilot" extension from Extensions
  3. Open Terminal (Spotlight → Terminal) and type: git clone https://github.com/tamirdresher/squad-for-kids.git
  4. In VS Code: File → Open Folder → squad-for-kids
  5. In the Terminal type: copilot and press Enter
  6. ⚡ Select the "Squad" agent from the menu — that's what activates the learning team

👨‍👩‍👧 For Parents — How It Works

Your child gets their own personal copy of the system. All progress is saved in your GitHub account — private and under your control.

🍴 Fork = Your Personal Copy

5 Simple Steps

  1. 🍴 Fork the Repository — Go to the original repo and click the "Fork" button in the top-right corner. This creates a personal copy in your account.
  2. ✅ Confirm you're on your Fork — Check that the URL shows your-username/squad-for-kids (not tamirdresher/squad-for-kids)
  3. 🚀 Create a Codespace from your Fork — Click "Code" → "Codespaces" → "Create codespace on main"
  4. 🤖 Select the Squad Agent — Open Copilot Chat (💬), select "squad" from the dropdown, and enable "Autopilot"
  5. 💬 Your child starts chatting! — Hey! I want to build a game!

📊 What Parents Can See

  • 📄 Student Profile (student-profile.json) — name, age, interests, XP, badges
  • 📁 Weekly Reports (.squad/reports/) — what was learned, strengths, areas to practice
  • 📋 Learning Plan (.squad/teaching-plan.md) — curriculum progress
  • 🔄 Commit History — every learning session is documented

🔄 Automatic Updates

When we add new features, templates, or learning content — you can sync them to your Fork:

  1. On your Fork page, look for the "This branch is X commits behind" banner
  2. Click "Sync fork" → "Update branch"
  3. ✅ Your child's personal data is preserved — only the learning content gets updated!

🔒 Privacy & Safety

  • 🔒 All data stays in your Fork — only you can see it
  • 👶 Minimal collection — only first name, age, and city
  • 🛡️ Age-appropriate content filtering for every age group
  • 📖 Open source — inspect every line of code yourself
