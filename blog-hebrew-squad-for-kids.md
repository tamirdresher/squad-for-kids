## 🤖 מה זה Squad? {#מה-זה-squad}

תארו לעצמכם שיש לכם **צוות שלם של עוזרים חכמים** שיושבים לידכם ועוזרים בכל מה שתרצו. זה Squad.

אני בניתי את Kids Squad כי הבת שלי ניגשה אליי פעם ואמרה "אבא, ChatGPT מבאס — הוא מסביר לי כאילו אני מבוגרת". היא הייתה צודקת. רוב כלי ה-AI מדברים בשפה של מבוגרים, מניחים ידע שאין לילדים, ולא ממש מבינים מה זה להיות בן 12 שרוצה לבנות משחק מגניב.

אז בניתי Squad לילדים. במקום עוזר AI אחד, אתם מקבלים **צוות שלם**:

| תפקיד | מה עושה | איך מדברים |
|--------|----------|------------|
| מורה | מסביר מושגים בשפה פשוטה | "מורה: תסביר לי מה זה משתנים" |
| מתכנת | כותב קוד איתך | "מתכנת: תעזור לי לבנות משחק" |
| מעצב | עוזר עם עיצוב | "מעצב: תעזור לי לעצב דף אישי" |
| בודק | בודק שהכל עובד | "בודק: תבדוק את הקוד שלי" |
| חוקר | עוזר בלימודים | "חוקר: יש לי מבחן בהיסטוריה מחר" |
| מומחה מחשבים | בעיות מחשב ובטיחות | "מומחה מחשבים: המחשב שלי איטי" |

**למה Squad ולא ChatGPT רגיל?**

- הצוות **מדבר עברית בצורה טבעית**
- הוא **מותאם לגיל שלכם** — ילד בן 10 מקבל הסברים אחרים מבן 15
- יש **פרויקטים מוכנים** שאפשר להתחיל מיד
- **ראלף** עוקב אחרי ההתקדמות שלכם ומעודד
- הכל **חינמי** — GitHub Copilot Free עובד בלי לשלם שקל

## ⚡ איך מתחילים ב-30 שניות {#איך-מתחילים}

### שלב 1: Fork

1. היכנסו ל: **github.com/tamirdresher/kids-squad-setup**
2. לחצו על כפתור **Fork** (למעלה מימין)
3. לחצו **Create fork**

מה זה Fork? זה כמו לצלם דף עבודה — מקבלים עותק משלכם שאפשר לשנות בלי לפגוע במקור.

### שלב 2: פתחו סביבת פיתוח

הכי קל — **GitHub Codespace** (בלי להתקין כלום!):

1. בדף ה-Fork שלכם, לחצו על **Code** (הכפתור הירוק)
2. עברו ללשונית **Codespaces**
3. לחצו **Create codespace on main**
4. חכו כדקה

Codespace זו סביבת תכנות שרצה בענן, ישר בדפדפן. לא צריך להתקין כלום. לוחצים, מחכים דקה, ומתכנתים.

**אפשרות נוספת — VS Code מקומי:**
1. הורידו VS Code (חינמי) מ-code.visualstudio.com
2. התקינו את התוסף GitHub Copilot
3. עשו Clone ל-Fork שלכם ופתחו ב-VS Code

### שלב 3: שלום!

1. פתחו את **Copilot Chat** (פאנל הצ'אט בצד)
2. כתבו: **שלום**
3. זהו! ה-AI יובילו אתכם צעד אחר צעד

Fork + Codespace + "שלום" = הצוות פעיל! כל הסיפור.

## 🆓 איך מקבלים Copilot בחינם {#copilot-בחינם}

### דרך 1: Copilot Free — לכולם (הכי מהיר)

GitHub Copilot Free זמין לכל מי שיש לו חשבון GitHub — בלי שום דרישות מיוחדות.

**מה מקבלים:**
- כ-2,000 השלמות קוד בחודש
- 50 הודעות צ'אט בחודש
- עובד ב-VS Code, Codespaces, ועל github.com
- תמיכה מלאה בעברית

**איך מפעילים:**
1. היכנסו ל-github.com וצרו חשבון (או התחברו)
2. היכנסו ל-github.com/features/copilot
3. לחצו **Get started for free**
4. זהו! Copilot פעיל

50 הודעות צ'אט בחודש מספיקות בהחלט לילד שמתכנת כמה פעמים בשבוע. שאלו שאלות מפורטות כדי לסחוט מכל הודעה כמה שיותר!

### דרך 2: Student Developer Pack — Copilot Pro בחינם לסטודנטים

אם אתם תלמידים מגיל 13, אפשר לקבל **Copilot Pro בחינם** — צ'אט ללא הגבלה! זה חלק מה-GitHub Student Developer Pack, שכולל כלים מקצועיים בשווי מאות דולרים.

#### מה צריך

- גיל 13 ומעלה
- חשבון GitHub
- לומדים כרגע בבית ספר
- אימות דו-שלבי (2FA) מופעל
- הוכחת סטטוס תלמיד (מייל בית ספר או מסמך)

#### המדריך שלב אחר שלב

**שלב 1: יצירת חשבון GitHub**

1. היכנסו ל-github.com/signup
2. הכניסו כתובת מייל (אם יש לכם מייל בית ספר — מצוין! זה מזרז את האימות)
3. בחרו שם משתמש וסיסמה
4. אמתו את המייל

**שלב 2: הפעלת אימות דו-שלבי (2FA) — חובה!**

בלי 2FA הבקשה לא תאושר.

1. היכנסו ל-Settings בפרופיל (לחצו על האווטאר → Settings)
2. לחצו על **Password and authentication**
3. תחת **Two-factor authentication**, לחצו **Enable**
4. בחרו שיטה: אפליקציית Authenticator (מומלץ) או SMS
5. שמרו את קודי השחזור במקום בטוח!

להורים: עזרו לילדים להגדיר 2FA. השתמשו באפליקציית Authenticator בטלפון שלכם.

**שלב 3: הכנת הוכחת סטטוס תלמיד**

תצטרכו אחד מאלה:
- מייל בית ספר — הדרך המהירה ביותר
- תעודת תלמיד — תמונה ברורה עם שם ותאריך תפוגה
- אישור לימודים ממזכירות בית הספר
- מערכת שיעורים עם השם ותאריכים עדכניים

המסמך חייב להראות את השם שלכם ואת שנת הלימודים הנוכחית.

**שלב 4: הגשת הבקשה**

1. היכנסו ל-education.github.com/pack
2. לחצו **Get your pack** → **Sign up for Student Developer Pack**
3. מלאו את הטופס — שם בית הספר, שנת סיום משוערת, הוכחה
4. לחצו **Submit**

**שלב 5: ממתינים לאישור**

בדרך כלל לוקח כמה ימים. GitHub שולח מייל כשמאושר. אם נדחית — קראו את ההערות וצרפו מסמך ברור יותר.

**שלב 6: נהנים!**

לאחר האישור מקבלים:
- Copilot Pro — הודעות צ'אט ללא הגבלה!
- קרדיטים בענן — Azure, DigitalOcean ועוד
- שם דומיין חינמי לאתר אמיתי
- כלים מקצועיים נוספים

## 💡 מה אפשר לבנות — דוגמאות אמיתיות {#מה-לבנות}

### גילאי 8-10: דף גיבור-על עם אנימציות (HTML)

כתבו ב-Copilot Chat:

```
מתכנת: תעזור לי לבנות דף על הגיבור-על שלי.
שם: ברק-הרוח
כוחות: מהירות-על ורעמים
אני רוצה אנימציה כשלוחצים עליו
```

Copilot יייצר HTML עם CSS ו-JavaScript — דף אינטרנט אמיתי שאפשר לפתוח בדפדפן!

### גילאי 11-13: משחק יריות בחלל ב-JavaScript

כתבו:

```
מתכנת: אני רוצה לבנות משחק חלל.
הספינה שלי יורה לייזרים.
אויבים נופלים מלמעלה.
כל פגיעה = 10 נקודות.
אחרי 100 נקודות עולים רמה.
```

Copilot יבנה את המשחק צעד אחר צעד — Canvas, אנימציות, ניקוד, אפקטי קול!

### גילאי 14+: ניתוח נתונים ב-Python

```
חוקר: אני רוצה לנתח נתוני מזג אוויר של תל אביב.
הראה לי גרף של טמפרטורות לפי חודש.
ואמור לי איזה חודש הכי חם.
```

### עזרה בשיעורי בית

Squad הוא לא רק לתכנות. הצוות עוזר גם בלימודים:

```
מורה: תסביר לי מה היה המרד הגדול. אני בכיתה ו'.
חוקר: יש לי מבחן במתמטיקה מחר על משוואות. תכין לי תרגולים.
מורה: תסביר לי מה זה פוטוסינתזה בשפה פשוטה עם דוגמאות.
מומחה מחשבים: איך מתקינים Python על המחשב?
מומחה מחשבים: האינטרנט לא עובד, מה לבדוק?
```

הצוות עוזר לכם להבין חומר ומכין תרגולים. הוא לא ייתן לכם תשובות להעתיק — כי ככה לא לומדים!

## 🎯 חווית ה-Squad {#חוויית-squad}

### ראלף — החבר הכי טוב של המתכנת הצעיר

ראלף הוא עוזר מיוחד שעוקב אחרי הפרויקטים שלכם. תחשבו עליו כחבר שתמיד שואל "מה נשמע? צריך עזרה?"

פקודות:

| פקודה | מה קורה |
|-------|----------|
| ralph start | ראלף מתחיל לעקוב |
| ralph status | מה קורה בפרויקט |
| ralph continue | חזרתם אחרי הפסקה — ראלף מזכיר איפה עצרתם |
| ralph stop | ראלף נרדם |

ראלף מדבר עברית, מעודד, ועוקב אחרי ההתקדמות אוטומטית.

### התאמה לגיל — הצוות גדל איתכם

Squad מתאים את עצמו לגיל שלכם. אין הגדרות לשנות — פשוט ספרו לצוות כמה אתם בני/בנות:

**גילאי 8-10 — צוות הכיף**
- 4 סוכנים: מורה, מתכנת, מעצב, מומחה מחשבים
- שפה פשוטה עם המון אימוג'ים
- פרויקטים קצרים: דפי גיבורי-על, ציורים, אנימציות

**גילאי 11-13 — צוות המשחקים**
- 5 סוכנים (+ בודק)
- משחקים, אתרים, בוטים ואפליקציות
- מתחילים ללמוד מושגי תכנות אמיתיים

**גילאי 14+ — הצוות המקצועי**
- 6+ סוכנים (צוות מלא)
- Full-stack, ניתוח נתונים, הכנה לבגרות
- פרויקטים מורכבים: מסדי נתונים, API ועוד

### עולמות שאפשר לבחור

Squad יודע לדבר בשפה של היקומים האהובים עליכם:

- מיינקראפט — "בואו נבנה מוד ב-JavaScript"
- הארי פוטר — "בואו נבנה אתר כובע המיון"
- מלחמת הכוכבים — "בואו נבנה סימולטור ספינת חלל"
- פוקימון — "בואו נבנה פוקידקס"

## סיכום — מתחילים עכשיו!

**מסלול מהיר (5 דקות):**
1. יוצרים חשבון GitHub בגithub.com/signup
2. עושים Fork ל-kids-squad-setup
3. פותחים Codespace
4. כותבים "שלום" ב-Copilot Chat
5. בונים משהו מגניב!

**מסלול מלא (שבוע) — לסטודנטים:**
1. יוצרים חשבון GitHub
2. מפעילים 2FA
3. מגישים בקשה ל-Student Developer Pack
4. ממתינים לאישור
5. Fork + Codespace + שלום + Copilot Pro ללא הגבלה!

### שאלות נפוצות

**אני בן/בת 10 — אפשר להשתמש?**
בהחלט! Copilot Free עובד בכל גיל. ה-Student Pack דורש גיל 13+. עד אז, 50 הודעות בחודש מספיקות לפרויקטים מגניבים.

**ההורים שלי לא מבינים בטכנולוגיה. זה בסדר?**
לגמרי! Fork + Codespace + שלום עובד בלי שום עזרה מההורים. אם צריכים עזרה עם 2FA, ביקשו מהם להתקין אפליקציית Authenticator — זהו.

**מה אם נגמרות הודעות הצ'אט?**
שלוש אפשרויות: 1) ממתינים לחודש הבא, 2) משתמשים ב-ChatGPT/Gemini/Claude בחינם, 3) מגישים בקשה ל-Student Developer Pack.

**האם זה בטוח לילדים?**
Squad רץ על התשתית של GitHub עם מנגנוני הגנה מובנים. מצב ההתאמה לגיל שומר על אינטראקציות מתאימות. הורים — תמיד מוזמנים לשבת לידם ולראות מה הם בונים. זה ממש כיף לצפות.
