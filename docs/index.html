🌟 מומלץ!

הדרך הקלה ביותר

  1. יש לכם חשבון GitHub? מעולה! אם לא — צרו חשבון בחינם (דקה אחת, כמו להירשם לאתר רגיל)
  2. 🖥️ חשוב: שלב זה צריך להיעשות ממחשב נייד או נייח — לא מטלפון או טאבלט.
    לחצו על הכפתור הירוק:
    ✨ פתחו את סקוואד לילדים
    💡 שימו לב: כשהדף נפתח, גללו למטה ולחצו על הכפתור הירוק "Create codespace" (צור סביבת עבודה).
    ⏱️ חכו דקה-שתיים עד שהמסך נטען לגמרי.
  3. חכו דקה עד שהמסך נטען (תראו ממשק עם טקסט — זה בסדר!)
  4. בתחתית המסך יש שורת טקסט — כתבו שם: copilot ולחצו Enter
  5. ⚡ חשוב! כשנפתח חלון הצ'אט, בחרו את הסוכן "Squad" מהרשימה למעלה. זה מה שמפעיל את צוות הדמויות!
    🔍 מחפשים למעלה בחלון הצ'אט תפריט שכתוב בו "Agent" או "סוכן" → בוחרים Squad
  6. תנו לילד את המקלדת! 🎮
🔄 חוזרים בפעם הבאה? לא צריך ליצור מחדש! היכנסו ל-github.com/codespaces ותמצאו שם את סביבת העבודה שכבר יצרתם. פשוט לחצו עליה.

🗺️ המסך באנגלית? אל תדאגו!

💬 דרך VS Code — Copilot Chat (ידידותי יותר)

במקום לכתוב בטרמינל השחור, אפשר לדבר עם הצוות דרך חלון צ'אט נוח יותר

  1. פתחו את סביבת העבודה (Codespaces או מקומי)
  2. בצד שמאל של המסך, לחצו על אייקון הצ'אט 💬 (או Ctrl+Shift+I)
  3. בחלון הצ'אט שנפתח, ודאו שנבחר הסוכן "Squad" בתפריט למעלה (Agent → Squad)
  4. עכשיו כתבו: hi והצוות יופיע!
  5. הצוות ידבר אתכם דרך הצ'אט — יותר נוח מהטרמינל!
💡 טיפ: Copilot Chat הוא כמו וואטסאפ — פשוט כותבים הודעה ומקבלים תשובה. לא צריך לזכור פקודות.

💻 התקנה על מחשב הילד (Windows)

אם אתם מעדיפים שהכל יהיה על המחשב של הילד — בלי אינטרנט אחרי ההתקנה

  1. הורידו והתקינו GitHub Desktop — לחצו "Download for Windows"
  2. התחברו עם חשבון ה-GitHub שלכם
  3. לחצו "Clone a Repository" → כתבו: tamirdresher/squad-for-kids → Clone
  4. הורידו והתקינו Node.js (לחצו על הכפתור הירוק הגדול)
  5. הורידו והתקינו GitHub Copilot CLI
  6. פתחו את התיקייה → קליק ימני → "פתח בטרמינל"
  7. כתבו: copilot ולחצו Enter
  8. ⚡ בחרו את הסוכן "Squad" מתפריט ה-Agent למעלה
  9. תנו לילד את המקלדת! 🎮

👨‍👩‍👧 מחשב ההורה — ניהול וצפייה בהתקדמות

ההורה מתקין על המחשב שלו ורואה דוחות התקדמות. הילד לומד, ההורה עוקב.

  1. עקבו אחרי שלבי ההתקנה למעלה (Codespaces או Windows)
  2. כתבו: copilot ואז בחרו את הסוכן "Squad", ואז כתבו: show me the parent report
  3. תראו דוח שבועי: מה הילד למד, איפה הוא חזק, מה צריך לתרגל
  4. כל הנתונים נשמרים ב-GitHub שלכם — רק אתם רואים
💡 טיפ: אפשר לפתוח את הפרויקט ב-GitHub ולראות את לוח המשימות — כמו לוח מודעות דיגיטלי עם כל מה שהילד עשה ומה הבא בתור.

🪟 Windows

  1. הורידו והתקינו את Visual Studio Code (תוכנה חינמית של מייקרוסופט)
  2. פתחו את התוכנה, לחצו על האייקון של המרובע בצד שמאל (Extensions), וחפשו "GitHub Copilot" — התקינו
  3. הורידו את Git והתקינו (לחצו Next עד הסוף)
  4. פתחו Terminal בתוך VS Code (תפריט Terminal → New Terminal)
  5. כתבו: git clone https://github.com/tamirdresher/squad-for-kids.git
  6. פתחו את התיקייה שנוצרה: File → Open Folder → squad-for-kids
  7. ב-Terminal כתבו: copilot ולחצו Enter
  8. ⚡ בתפריט למעלה בחרו את הסוכן "Squad" — זה מה שמפעיל את צוות הלמידה

🍎 Mac

  1. הורידו והתקינו את Visual Studio Code
  2. התקינו את התוסף "GitHub Copilot" מתוך Extensions
  3. פתחו Terminal (Spotlight → Terminal) וכתבו: git clone https://github.com/tamirdresher/squad-for-kids.git
  4. ב-VS Code: File → Open Folder → squad-for-kids
  5. ב-Terminal כתבו: copilot ולחצו Enter
  6. ⚡ בתפריט למעלה בחרו את הסוכן "Squad" — זה מה שמפעיל את צוות הלמידה
