# Work Routing

## Template Development

| Task | Assigned To | Notes |
|------|-------------|-------|
| New squad template | Maria (lead) + Ken | Must pass Dr. Sarah safety review |
| Template content | Emma (lead) + Maria | Ken reviews for engagement |

## Issue Triage

- **Bug reports** → Sal (technical), Dr. Sarah (safety-related)
- **Feature requests** → Ken (educational), Emma (engagement)
- **Content requests** → Maria (curriculum), Emma (format)
- **Safety concerns** → Dr. Sarah (immediate priority)
- **New language support** → Maria (framework), community contributors (content)

## Review Requirements

All templates require:
1. ✅ Pedagogical review (Maria)
2. ✅ Engagement review (Ken or Emma)
3. ✅ Safety review (Dr. Sarah)
4. ✅ Technical review (Sal)
