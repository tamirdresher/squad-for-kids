# 🍕 משחק שברים — Fractions Game

משחק אינטראקטיבי ללמידת שברים בעברית, מתאים לילדים בגילאי 8–14.

## 🎮 מה יש במשחק?

- **3 רמות קושי** — קל, בינוני, ומאתגר
- **ויזואליזציות SVG** — עיגולי פיצה צבעוניים שמסבירים שברים
- **שאלות רב-ברירה** — עם ניקוד ומעקב חיים
- **הודעות עידוד בעברית** — "כל הכבוד! 🌟", "אל תתייאש/י! 💪"
- **מסך סיום עם כוכבים** — ⭐⭐⭐

## 📚 מה לומדים?

| רמה | נושא |
|-----|------|
| רמה 1 🟢 | מה זה שבר? מונה ומכנה |
| רמה 2 🟡 | השוואת שברים (גדול/קטן) |
| רמה 3 🔴 | חיבור שברים עם מכנה זהה |

## 🚀 איך משחקים?

פשוט פתחו את `index.html` בדפדפן — אין צורך בהתקנה או חיבור לאינטרנט!

```
starter-projects/fractions-game/index.html
```

## 🛠️ פרטים טכניים

- **HTML/CSS/JS** — בלי תלות חיצונית
- **RTL** — ממשק מלא בעברית, מימין לשמאל
- **Responsive** — עובד על מחשב וטאבלט
- **SVG** — גרפיקה וקטורית לשברים, נוצרת דינמית בקוד

## 💡 טיפים למורים והורים

- המשחק מתאים לחיזוק חומר שנלמד בכיתה ג'–ו'
- כל שאלה כוללת הסבר לאחר המענה
- אפשר לשחק כמה פעמים — השאלות מגיעות בסדר אקראי
