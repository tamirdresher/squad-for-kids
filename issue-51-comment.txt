### VULNERABILITY SUMMARY

**CVE-2026-24512** (CVSS 8.8): Remote code execution via Ingress path injection
- **Attack Vector:** Malicious `rules.http.paths.path` field → nginx configuration injection → RCE in controller pod
- **Impact:** Cluster-wide secret exfiltration, full cluster compromise, persistent backdoor
- **Affected Versions:** ingress-nginx < v1.13.7, < v1.14.3

**Additional Threats:**
- CVE-2025-1974 (RCE via annotation abuse)
- CVE-2026-24514 (DoS via admission controller flooding)
- Unauthenticated heartbeat endpoint DDoS risk

### EXPLOITABILITY: HIGH

**Attack Scenarios:**
1. **External Attacker:** Submit malicious Ingress via compromised CI/CD → RCE → secrets exfiltration → cluster takeover
2. **Lateral Movement:** Compromised tenant pod → internal ingress-controller endpoint → DDoS or Ingress creation → exploitation
3. **Supply Chain:** Trojanized Helm chart with malicious Ingress → automated deployment → immediate compromise

**Without Network Policies or WAF:** Any initial foothold = full cluster compromise
