## 🔧 סקריפטים שעובדים בלי AI

הסקריפטים האלה עובדים גם כש-Copilot לא זמין:

### ⏱️ טיימר לימודים (Pomodoro)
```bash
# פתח בדפדפן — לא צריך AI!
open starter-projects/html-fun/index.html
```

### 📊 ניתוח נתונים
```bash
# הרץ את סקריפט ה-Python — עובד לבד!
python starter-projects/python-science/main.py
```

### 🎮 משחק
```bash
# פתח את המשחק — עובד בלי שום דבר!
open starter-projects/js-games/index.html
```

### 🤖 ראלף
```bash
# ראלף עובד בלי AI!
pwsh ralph-kids.ps1 -Command "סטטוס"
```

## 💡 טיפים לחסוך שיחות

1. **שאלות ארוכות ומפורטות** — שיחה אחת עם הרבה פרטים = תוצאה טובה יותר מ-5 שיחות קצרות
2. **תשתמש בקבצים שנוצרו** — סיכומים, תבניות, ודפי עזרה לא צורכים שיחות
3. **תריץ סקריפטים** — הכלים ב-`starter-projects/` עובדים בלי AI
4. **תתכנן לפני** — תחשוב מה לשאול לפני שפותחים שיחה
