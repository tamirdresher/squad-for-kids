### 🧰 מה יותקן על המחשב

אל דאגה — כל הכלים האלה חינמיים ובטוחים. סקריפט ההתקנה מטפל בהכל אוטומטית, אבל הנה הסבר קצר על כל אחד:

| כלי | מה הוא עושה | בשפה פשוטה |
|-----|-------------|------------|
| 🏫 **VS Code** | עורך טקסט חינמי של מיקרוסופט | ה"כיתה" שבה הילד ילמד — סביבת הלמידה הראשית |
| 🧠 **GitHub Copilot** | עוזר חכם מבוסס בינה מלאכותית | ה"מוח" של הצוות — חינם לשימוש אישי |

> 🔒 **לגבי אבטחה:** כל הכלים הם קוד פתוח של חברות מוכרות (מיקרוסופט, GitHub). אין פרסומות, אין מעקב, אין תשלום.

## שלב 1: יצירת חשבון GitHub

> ⏭️ **כבר יש לכם חשבון GitHub?** מעולה! [דלגו לשלב 2](#שלב-2-עשו-fork-לריפו).

GitHub הוא אתר שבו מפתחים שומרים קוד — וגם המקום שבו ההתקדמות של הילד נשמרת בצורה בטוחה ומאורגנת.

### 📝 יוצרים חשבון:

1. **פתחו את הדפדפן** והיכנסו ל-[github.com/signup](https://github.com/signup)

2. **הכניסו כתובת אימייל** — השתמשו באימייל **שלכם** (של ההורה)
   > 📧 האימייל משמש לאימות ולקבלת דוחות התקדמות

3. **בחרו סיסמה חזקה** — לפחות 15 תווים, או 8 תווים עם מספר ואות
   > 🔒 טיפ: השתמשו במשפט שקל לזכור, למשל: `הילד-שלי-לומד-2025!`

4. **בחרו שם משתמש** — זה מה שיופיע בכתובת הריפו
   > 💡 למשל: `cohen-family` או `maya-learns`

5. **אמתו שאתם לא רובוט** 🤖 — פתרו את החידה הקטנה

6. **לחצו "Create account"**

7. **בדקו את האימייל** — תקבלו קוד אימות. הכניסו אותו באתר.

8. ✅ **מזל טוב!** יש לכם חשבון GitHub!

> ⚠️ **הערה חשובה לגבי גיל:** על פי תנאי השימוש של GitHub, ילדים מתחת לגיל 13 צריכים להשתמש בחשבון של ההורה. זו בדיוק הסיבה שאנחנו מבקשים שההורה ייצור את החשבון. הילד עובד בתוך סביבה שההורה שולט בה.

## שלב 2: עשו Fork לריפו

### 🍴 מה זה Fork?

בואו נסביר: ה"ריפו" (Repository) הוא כמו **ספר מוכן** עם כל תכנית הלימודים. כשאתם עושים **Fork**, אתם יוצרים **עותק אישי** של הספר לילד שלכם. הילד יכול לכתוב בספר שלו, לסמן דפים, לקפל פינות — בלי לפגוע בספר המקורי.

### 📝 איך עושים Fork:

1. **היכנסו ל-GitHub** — ודאו שאתם מחוברים לחשבון שלכם

2. **לכו לריפו המקורי:** [github.com/tamirdresher/squad-for-kids](https://github.com/tamirdresher/squad-for-kids)

3. **לחצו על כפתור "Fork"** 🍴
   > 📍 הכפתור נמצא בפינה הימנית העליונה של הדף, ליד כפתורי ⭐ Star ו-👁️ Watch.
   > הוא נראה כמו חץ מתפצל.

4. **בדף "Create a new fork":**
   - **Owner** — בחרו את חשבון ה-GitHub שלכם
   - **Repository name** — השאירו כ-`squad-for-kids`
   - ✅ סמנו **"Copy the main branch only"**
   - לחצו **"Create fork"** 🎉

5. **חכו כמה שניות** — תועברו אוטומטית ל-Fork החדש שלכם

### ✅ איך יודעים שזה עבד?

הסתכלו על הכתובת בראש הדף. צריך להיות כתוב:

```
שם-המשתמש-שלכם / squad-for-kids
forked from tamirdresher/squad-for-kids
```

> ⚠️ **טעות נפוצה:** ודאו שאתם על **הFork שלכם** ולא על `tamirdresher/squad-for-kids`. אם תתקינו מהריפו המקורי, ההתקדמות של הילד לא תישמר בחשבון שלכם!

## שלב 3: הריצו את סקריפט ההתקנה

> 🪄 סקריפט ההתקנה עושה את כל העבודה בשבילכם — מתקין את כל הכלים, מגדיר הכל, ומוריד את הריפו למחשב. פשוט העתיקו והדביקו!

### 🪟 Windows

#### פתיחת PowerShell:

1. **לחצו על מקש Windows** ⊞ (המקש עם הלוגו של Windows, למטה משמאל)
2. **הקלידו:** `PowerShell`
3. **לחצו ימני** על "Windows PowerShell" ובחרו **"הפעל כמנהל מערכת"** (Run as Administrator)
   > 🔑 זה חשוב — חלק מההתקנות דורשות הרשאות מנהל
4. **אם צצה הודעה** "האם להרשות לאפליקציה הזו לבצע שינויים?" — לחצו **"כן"**

#### הרצת סקריפט ההתקנה:

5. **העתיקו את הפקודה הזו** (לחצו על הטקסט ואז `Ctrl+C`):

```powershell
irm https://raw.githubusercontent.com/tamirdresher/squad-for-kids/main/setup.ps1 | iex
```

6. **הדביקו ב-PowerShell** (`Ctrl+V`) ולחצו **Enter**

#### 🔄 מה קורה עכשיו?

הסקריפט יריץ את השלבים האלה (יכול לקחת 5-10 דקות):

| שלב | מה קורה | מה תראו |
|-----|---------|---------|
| 1️⃣ | בדיקת כלים קיימים | `✓ Checking installed tools...` |
| 2️⃣ | התקנת Git | `📥 Installing Git...` |
| 3️⃣ | התקנת Node.js | `📥 Installing Node.js...` |
| 4️⃣ | התקנת VS Code | `📥 Installing VS Code...` |
| 5️⃣ | התקנת GitHub CLI | `📥 Installing GitHub CLI...` |
| 6️⃣ | התחברות ל-GitHub | `🔗 Logging into GitHub...` (ייפתח דפדפן) |
| 7️⃣ | שכפול הריפו | `📂 Cloning your fork...` |
| 8️⃣ | התקנת תוספי VS Code | `🧩 Installing extensions...` |
| 9️⃣ | פתיחת VS Code | `🚀 Opening VS Code...` |

> 💡 **אל תבהלו מהטקסט הרבה שרץ על המסך!** זה נורמלי. הסקריפט מספר לכם מה הוא עושה בכל רגע.

> 🌐 **שלב ההתחברות:** באמצע ההתקנה ייפתח חלון דפדפן שמבקש מכם להתחבר ל-GitHub. הכניסו את שם המשתמש והסיסמה, ואשרו את ההרשאות.

### 🍎 macOS

#### פתיחת Terminal:

1. **לחצו `Cmd+Space`** כדי לפתוח את Spotlight (חיפוש)
2. **הקלידו:** `Terminal`
3. **לחצו Enter** — ייפתח חלון שחור עם שורת פקודה
   > 💡 Terminal הוא כמו "שורת הפקודה" של Mac. לא צריך לפחד ממנו!

#### הרצת סקריפט ההתקנה:

4. **העתיקו את הפקודה הזו:**

```bash
curl -fsSL https://raw.githubusercontent.com/tamirdresher/squad-for-kids/main/setup.sh | bash
```

5. **הדביקו ב-Terminal** (`Cmd+V`) ולחצו **Enter**

6. **אם המערכת מבקשת סיסמה** — הכניסו את סיסמת המחשב שלכם (הסיסמה לא תופיע על המסך, זה נורמלי!)

7. ⏱️ חכו 5-10 דקות עד שההתקנה תסתיים

> 💡 **הערה ל-macOS:** אם מופיעה הודעה על Xcode Command Line Tools, אשרו את ההתקנה — זה כלי בסיסי שנדרש.

#### הרצת סקריפט ההתקנה:

4. ⏱️ חכו 5-10 דקות

### 4.4 התקנת GitHub CLI

<details>
<summary>🪟 <strong>Windows</strong></summary>

פתחו PowerShell והריצו:
```powershell

```

</details>

<details>
<summary>🍎 <strong>macOS</strong></summary>

פתחו Terminal והריצו:
```bash
brew install gh
```
> 💡 אין לכם Homebrew? התקינו קודם: `xcode-select --install` ואז `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

<details>
<summary>🐧 <strong>Linux</strong></summary>

```bash
(type -p wget >/dev/null || (sudo apt update && sudo apt-get install wget -y)) \
  && sudo mkdir -p -m 755 /etc/apt/keyrings \
  && out=$(mktemp) && wget -nv -O$out https://cli.github.com/packages/githubcli-archive-keyring.gpg \
  && cat $out | sudo tee /etc/apt/keyrings/githubcli-archive-keyring.gpg > /dev/null \
  && sudo chmod go+r /etc/apt/keyrings/githubcli-archive-keyring.gpg \
  && echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null \
  && sudo apt update \
  && sudo apt install gh -y
```

</details>

✅ בדיקה: `gh --version`

## שלב 5: מתחילים ללמוד! 🎉

### 🚀 הפעלת הצוות בפעם הראשונה

1. **פתחו VS Code** — אם הוא לא נפתח אוטומטית, חפשו אותו בתפריט ההתחלה או ב-Spotlight

2. **פתחו את תיקיית הריפו** — אם לא נפתחה אוטומטית:
   - לחצו `Ctrl+K Ctrl+O` (או `Cmd+K Cmd+O` ב-Mac)
   - נווטו לתיקייה `squad-for-kids` ובחרו אותה

3. **פתחו Copilot Chat** 💬
   - לחצו `Ctrl+Alt+I` (או `Cmd+Alt+I` ב-Mac)
   - או לחצו על אייקון הצ'אט 💬 בסרגל הצד

4. **בחרו את סוכן squad** 🤖
   - בתחתית חלון הצ'אט, תראו תפריט נפתח
   - בחרו **"squad"** מהרשימה

5. **הפעילו מצב Autopilot** 🚗
   - לחצו על **"Autopilot (Preview)"** בתחתית חלון הצ'אט
   - זה מאפשר לצוות לעבוד באופן עצמאי

6. **תנו לילד את המקלדת!** ⌨️ שהילד יכתוב:

```
היי! אני רוצה ללמוד! 🎉
```

### 🎬 מה קורה בפגישה הראשונה?

הצוות יעשה את הכל אוטומטית:

1. 👋 **מקבל את פני הילד** — ברכה חמה ואישית
2. 📝 **שואל את השם** — "מה שמך? ספר/י לי קצת על עצמך!"
3. 🎂 **שואל על הגיל/הכיתה** — כדי להתאים את רמת הלימודים
4. 🏙️ **שואל על העיר** — כדי לזהות אוטומטית את תכנית הלימודים
5. 🎮 **נותן לבחור ערכת נושא** — הארי פוטר, מיינקראפט, גיבורי-על, או יותר!
6. 👥 **יוצר את צוות הלמידה** — כל סוכן מציג את עצמו
7. 🎯 **מציע פרויקט ראשון** — משהו כיפי שמתאים לתחומי העניין של הילד

> 🎉 **טיפ:** שבו ליד הילד בפגישה הראשונה. זו חוויה מגניבה לשניכם!

### 🔄 חוזרים בפעם הבאה

בפעמים הבאות, פשוט:
1. פתחו VS Code
2. פתחו Copilot Chat (`Ctrl+Alt+I`)
3. בחרו את סוכן "squad"
4. הפעילו Autopilot
5. הילד כותב `היי!` — והצוות ממשיך מאיפה שהפסיקו! 🎯

## 🔧 פתרון בעיות

### 🪟 "PowerShell לא מריץ את הסקריפט"

**הבעיה:** מופיעה הודעת שגיאה על ExecutionPolicy.

**הפתרון:** הריצו את הפקודה הזו לפני סקריפט ההתקנה:
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
ואז נסו שוב את סקריפט ההתקנה.

> 💡 **מה זה אומר?** Windows חוסם סקריפטים מהאינטרנט כברירת מחדל כדי להגן עליכם. הפקודה הזו אומרת ל-Windows: "בסדר, סמוך עלי."

---

### ❌ "מופיעה שגיאת 'not recognized'"

**הבעיה:** הרצתם פקודה כמו `git` או `node` ומופיע: `'git' is not recognized as an internal or external command`

**הפתרון:**
1. **סגרו את PowerShell / Terminal**
2. **פתחו מחדש** — כלים חדשים שהותקנו צריכים חלון חדש כדי שיזוהו
3. נסו שוב

**אם עדיין לא עובד:**
- **Windows:** הפעילו מחדש את המחשב 🔄
- **macOS/Linux:** הריצו `source ~/.bashrc` או `source ~/.zshrc`

---

### 🧩 "VS Code לא מראה Copilot"

**הבעיה:** אין אייקון צ'אט, או ש-Copilot לא מגיב.

**הפתרון:**
1. ודאו שתוסף **GitHub Copilot** מותקן:
   - לחצו `Ctrl+Shift+X` → חפשו "GitHub Copilot" → ודאו שכתוב "Installed"
2. ודאו שאתם **מחוברים ל-GitHub** ב-VS Code:
   - לחצו על אייקון האדם 👤 בפינה השמאלית התחתונה
   - אם כתוב "Sign in", לחצו והתחברו
3. **הפעילו מחדש** את VS Code (`Ctrl+Shift+P` → הקלידו "Reload Window" → Enter)

---

### ⏸️ "ההתקנה נעצרת באמצע"

**הבעיה:** הסקריפט נתקע ולא מתקדם.

**הפתרון:**
1. לחצו `Ctrl+C` כדי לעצור את הסקריפט
2. הריצו את הסקריפט שוב — הוא ידלג על מה שכבר מותקן
3. אם נתקע שוב, נסו את ה[התקנה הידנית](#שלב-4-התקנה-ידנית-חלופה)

---

### 🤖 "לא מוצא את סוכן squad"

**הבעיה:** בתפריט הנפתח של Copilot Chat לא מופיע "squad".

**הפתרון:**
1. ודאו שפתחתם את **תיקיית squad-for-kids** ב-VS Code (ולא קובץ בודד)
2. ודאו שתוסף **GitHub Copilot** מותקן ומעודכן
3. נסו לסגור ולפתוח מחדש את VS Code
4. אם עדיין לא מופיע — לחצו `Ctrl+Shift+P`, הקלידו "Reload Window", ולחצו Enter

---

### 🗑️ "הילד מחק קובץ בטעות"

**הבעיה:** קובץ חשוב נמחק.

**אל פאניקה!** 😌 Git שומר על הכל. הריצו ב-Terminal:

```bash
git restore שם-הקובץ
```

למשל, אם הילד מחק את `student-profile.json`:
```bash
git restore student-profile.json
```

**אם כמה קבצים נמחקו:**
```bash
git restore .
```

> 💡 **זו בדיוק הסיבה ש-Git מותקן!** הוא כמו "Ctrl+Z" למחשב שלם.

---

### 🔄 "איך מעדכנים לגרסה חדשה?"

כשמתווספות תכונות חדשות ל-Squad for Kids:

**דרך האתר:**
1. היכנסו לFork שלכם ב-GitHub
2. תראו באנר: **"This branch is X commits behind tamirdresher:main"**
3. לחצו **"Sync fork"** → **"Update branch"**

**דרך שורת הפקודה:**
```bash
cd squad-for-kids
git pull upstream main
```

> 🔒 **לא לדאוג:** המידע האישי של הילד (פרופיל, דוחות, היסטוריית למידה) לא ייפגע. רק תכנית הלימודים והתכונות מתעדכנות!
